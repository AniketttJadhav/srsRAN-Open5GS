# srsRAN-Open5GS



\# srsRAN Project + Open5GS — Midterm Report



\## Overview

This repository contains the midterm report and supporting files for the 

5G network implementation using srsRAN Project and Open5GS on a remote 

SSH server.



\## Repository Structure

```

srsRAN-Open5GS/

├── report/

│   ├── main.tex              # Main LaTeX file

│   ├── references.bib        # Bibliography

│   ├── figures/              # Images and diagrams

│   └── sections/

│       ├── introduction.tex

│       ├── methodology.tex

│       └── results.tex

├── scripts/                  # Setup and config scripts

├── matlab/                   # MATLAB analysis code

└── README.md

```



\## How to Compile the Report

Make sure you have a LaTeX distribution installed (e.g. MiKTeX or TeX Live).

```bash

cd report

pdflatex main.tex

bibtex main

pdflatex main.tex

pdflatex main.tex

```

\- Server OS: Ubuntu
