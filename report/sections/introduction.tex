\section{Introduction}
This report presents the midterm progress of implementing a 5G network 
using srsRAN Project and Open5GS. The srsRAN Project provides an open-source 
5G RAN (Radio Access Network) implementation, while Open5GS serves as the 
5G Core Network (5GC).

The goal of this project is to set up a fully functional 5G network on a 
server environment, analyze its performance, and document the configuration 
and results obtained.

\subsection{Objectives}
\begin{itemize}
    \item Install and configure Open5GS as the 5G core network
    \item Install and configure srsRAN Project as the 5G RAN
    \item Establish end-to-end connectivity between RAN and Core
    \item Analyze and document network performance
\end{itemize}
